# 스토리보드 명세서 (UI Description)

> **Notice:** 본 문서는 초기 기획 단계의 [**초안 스케치**](/docs/03-StoryBoard-Sketch/StoryBoard-Sketch.pdf)를 기반으로 작성되었으며, 구체적인 앱 디자인 시 앱 디자인 가이드라인(Design Guidelines)에 따라 일부 수정될 수 있음을 알려드립니다.

## 1. 메인 화면 (Home) [Page 1]
앱의 첫 진입 화면으로 사용자 상태, 미션, 퀵 메뉴를 제공합니다.

### UI 구성요소
* **Top Bar (상단 바)**
    * **로고:** '쓰레기 박사' 텍스트 및 로고 (등급 아이콘 포함)
    * **알림:** 알림 아이콘 (🔔)
    * **사용자 정보:** "[닉네임]님! 오늘도 분리수거 잘하고 계시나요?" 문구 및 프로필 이미지

* **Body (본문)**
    * **오늘의 미션 카드:**
        * 타이틀: "오늘의 미션카드"
        * 미션 리스트:
            1. 오늘 N번 분리배출하기 (체크박스 포함)
            2. 걸음 수 0000보 걷기 (연동 데이터 표시)
        * 포인트 정보: 보유 포인트 (XXX point) 및 포인트/사용 내역 확인 버튼 (클릭 시 이동)
    * **Quick Link (바로가기 메뉴):**
        * 아이콘 그리드 형태: 수동 검색, 지도(Map), 자원봉사, 설정 등
    * **커뮤니티 미리보기:**
        * 섹션 타이틀: "커뮤니티" (더보기 `>` 버튼)
        * 피드 리스트: 사용자 작성 게시글 미리보기 (제목, 이미지, 작성 시간)
            * 예: "플라스틱을 줄였어요!", "분리배출 꿀팁 공유"
    * **상점/대리수거 미리보기:**
        * 하단에 상점 및 대리수거 서비스로 이동하는 배너 또는 섹션 배치

* **Bottom Navigation Bar (하단 탭 바)**
    1.  **자원봉사** (손 하트 아이콘)
    2.  **커뮤니티** (사람 3명 아이콘)
    3.  **카메라/홈** (중앙 강조된 카메라 아이콘 - 쓰레기 인식)
    4.  **대리수거** (쓰레기통 아이콘)
    5.  **상점** (장바구니 아이콘)

## 2. 커뮤니티 화면 [Page 2, 3]
사용자 간 소통, 챌린지 참여, 랭킹을 확인하는 공간입니다.

### UI 구성요소
* **Tab Menu (상단 탭):** [피드] / [챌린지] / [랭킹]
    * 탭 선택 시 하단 콘텐츠 변경 (색상 변화로 활성 상태 표시)

* **Tab 1: 피드 (Feed)**
    * **게시글 리스트:**
        * 썸네일 이미지 (좌측)
        * 제목, 내용 요약
        * 작성자 닉네임, 작성 날짜, 조회수/좋아요 수
    * **플로팅 버튼 (FAB):** 우측 하단 글쓰기 버튼 (`+` 또는 연필 아이콘)

* **Tab 2: 챌린지 (Challenge)**
    * **챌린지 리스트:**
        * 카드 형태: 챌린지 이미지, 제목 ("플라스틱 줄이기 챌린지" 등)
        * 상태 표시: 참여 중 / 참여 가능 / 종료
        * 참여 인원 및 달성률 표시

* **Tab 3: 랭킹 (Ranking)**
    * **랭킹 필터:** 일간 / 주간 / 월간 탭
    * **내 랭킹:** 상단에 내 현재 순위와 포인트 고정 표시
    * **랭킹 리스트:** 1위~N위 사용자 리스트 (순위, 닉네임, 포인트, 프로필 사진)

## 3. 쓰레기 인식 및 결과 화면 [Page 4, 5]
AI 카메라를 통해 폐기물을 인식하고 결과를 확인하는 핵심 기능입니다.

### 화면 3-1: 인식 실패/오류 [Page 4]
* **상태:** AI가 사물을 명확히 인식하지 못했을 때
* **팝업/모달:**
    * 아이콘: 경고(⚠️) 또는 눈 아이콘
    * 메시지: "정확한 인식이 불가합니다."
    * **버튼 1:** [재촬영] (카메라 다시 실행)
    * **버튼 2:** [수동 검색] (직접 품목 입력 화면으로 이동)

### 화면 3-2: 인식 성공 [Page 5 상단]
* **상태:** AI 인식 성공 시
* **팝업/모달:**
    * 메시지: "인식 완료!"
    * 결과 텍스트: **[ 품목명 ]** (예: 페트병)
    * **버튼 1:** [배출 가이드 조회]
    * **버튼 2:** [닫기]

### 화면 3-3: 배출 가이드 상세 [Page 5 하단]
* **타이틀:** "[품목명]의 배출 가이드"
* **가이드 내용:**
    1.  내용물 비우기
    2.  라벨 제거하기
    3.  압착하기
    4.  뚜껑 닫아서 배출 (구체적 단계별 텍스트 및 아이콘)
* **하단 버튼:**
    * **[분리 인증하기]:** 인증샷 촬영 화면으로 이동 (포인트 적립 연계)
    * **[닫기]:** 메인 또는 카메라 화면으로 복귀

## 4. 대리수거 화면 [Page 6, 7]
대리수거 요청 목록을 확인하고, 직접 요청을 등록하는 기능입니다.

### 화면 4-1: 요청 목록 (수거자 뷰) [Page 6]
* **타이틀:** "대리수거 요청"
* **리스트 아이템:**
    * 품목: (예: 플라스틱, 종이, 책)
    * 장소: (예: OO 오피스텔 301동) - *상세 주소는 승낙 전 일부 마스킹 가능*
    * 날짜: 2025.11.11
    * 포인트/보상: 수행 시 획득 가능한 포인트
* **상세 보기 (클릭 시):**
    * 지도/위치 표시
    * 요청 사진
    * **[수거하기] 버튼:** 클릭 시 매칭 성사 및 상세 주소 공개

### 화면 4-2: 요청하기 (요청자 뷰) [Page 7]
* **타이틀:** "대리수거 요청하기"
* **입력 폼:**
    1.  **장소:** 주소 검색 및 상세 주소 입력 (도로명 주소)
    2.  **품목:** 카테고리 선택 (체크박스 또는 드롭다운)
    3.  **사진:** [사진 촬영] 또는 [앨범 선택] 버튼 (폐기물 사진 첨부 필수)
    4.  **포인트 설정:** (선택 사항 - 추가 팁 등)
* **하단 버튼:** [제출하기] 또는 [요청 등록]

## 5. 자원봉사 화면 [Page 8, 9]
환경 봉사활동(플로깅 등)을 모집하거나 참여하는 커뮤니티 기능입니다.

### 화면 5-1: 봉사활동 목록 [Page 8]
* **타이틀:** "봉사활동 모집"
* **상단 버튼:** [모집하기] (글쓰기)
* **리스트 아이템:**
    * 썸네일 이미지 (지도 또는 활동 사진)
    * 제목: "플로깅 하실 분 모집합니다!"
    * 정보: 지역(인천시), 날짜(2025.12.31), 인원(2/8)
    * 상태: 모집중 / 마감임박 / 마감

### 화면 5-2: 봉사활동 상세 [Page 8 하단]
* **내용:** 작성자 프로필, 상세 설명 (위치, 집합 장소, 시간 등)
* **하단 버튼:** [참여하기] (신청 시 인원 카운트 증가 및 알림 설정)

### 화면 5-3: 모집글 작성 [Page 9]
* **타이틀:** "봉사 모집하기"
* **입력 폼:**
    * **제목:** 활동명 입력
    * **위치:** 활동 장소 지정
    * **날짜:** 활동 일시 선택 (캘린더)
    * **모집 인원:** 숫자 입력 (예: 8명)
    * **사진 첨부:** 활동 관련 이미지
    * **본문:** 상세 내용 입력 (준비물, 주의사항 등)
* **상단 버튼:** [게시하기] (등록 완료)

## 6. 공통 네비게이션 (GNB)
모든 화면 하단에 고정된 메뉴바입니다.

* **자원봉사:** 봉사활동 모집/참여 화면으로 이동
* **커뮤니티:** 피드/챌린지/랭킹 화면으로 이동
* **카메라 (중앙):** AI 쓰레기 인식 카메라 실행 (메인 기능)
* **대리수거:** 대리수거 매칭 서비스 화면으로 이동
* **상점:** 포인트 사용 상점 화면으로 이동
